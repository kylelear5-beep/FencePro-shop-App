CMM Machine Routing Guides

CMM Machine Operations

Overview: This guide provides technical specifications and operational steps for the CMM post-routing machine used for Vinyl and Aluminum fabrication.

Precision Alphanumeric Routing (ABC Wizard)

  • Router Bit: Use an 1/8-inch router bit for alphanumeric routing for clean cuts.
  • Layout: Options include Top-to-Bottom, Left-to-Right, or rotated 90-degrees for signs.
  • Spacing: Leave spacing at zero for natural look; use spacebar for manual gaps.

Angles Wizard

  • Scope: Applies adjustments to all holes on the current face.
  • Rise vs Side: Rise Angle elongates holes; Side Angle makes them wider.
  • Pivot: Use the middle pivot point to stretch the rise angle equally.

Configuration & Operations

  • End Spacing: Distance from top edge of post to the top of the highest hole.
  • Importing: Navigate to File > Import Part Data. Select the index file ("master key").
  • Dry Run: Use to mimic movements without plunging bit. Always test new programs.
  • T-Rail Adapter: Use for T-shaped handrails; remember this offsets home position.

Safety & E-Stops

  • Software Stop: Use on-screen button for bit breaks or simple pauses.
  • Mechanical E-Stop: Use the red cabinet button for mechanical issues or reaching inside.
