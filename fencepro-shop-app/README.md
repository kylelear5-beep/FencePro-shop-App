# Fence-Pro Command Center
### Superior Fence & Rail Shop Operations Platform

Fence-Pro is a high-efficiency production workstation designed for the shop floor. It integrates the 20-year veteran knowledge of **Big Bob** with local SOPs and operational tools into a single, cohesive brand-aligned application.

---

## 🛠 Features

### 1. Operations Dashboard
- **Real-Time Module View**: Tracks quality compliance, veteran tech years, and Big Bob's connectivity status.
- **Dynamic SOP Counter**: Automatically scans and counts live documents in the library.
- **Brand Consistency**: Fully aligned with the `superiorfenceandrail.com` corporate identity using the official Oswald/Roboto typography and Maroon/Green/Beige palette.

### 2. Big Bob (Shop Boss Consultation)
- **Veteran Engine**: Powered by Gemini 2.0-Flash to handle complex procedure queries.
- **Persona-First Interaction**: Big Bob maintains his original, no-nonsense foreman personality. 
- **Operational Intelligence**: Provides specific advice on loading, staging color codes, and PPE requirements.

### 3. Document Library (SOPs)
- **Searchable Access**: Instant access to all Superior Standard Operating Procedures.
- **Centralized Linkage**: Direct integration with production PDFs and training guides.

---

## 🚀 Getting Started

### Prerequisites
- Node.js installed
- A valid Google Gemini API Key in the `.env` file (`GEMINI_API_KEY=your_key_here`)

### Installation & Launch
1. Install dependencies:
   ```bash
   npm install
   ```
2. Run the Command Center:
   ```bash
   npm run dev
   ```
3. Open in your browser:
   ```
   http://localhost:3000
   ```
